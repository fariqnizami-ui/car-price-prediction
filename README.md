# Car Price Prediction App 🚗💰

A machine learning web application that predicts the selling price of used cars based on features like present price, kilometers driven, fuel type, and age.

## 🚀 Features
* **Predictive Modeling:** Uses a Random Forest Regressor to estimate car value.
* **Interactive UI:** Built with Streamlit for an easy-to-use web interface.
* **Feature Engineering:** Automatically calculates car age and handles categorical encoding.

## 🛠️ Tech Stack
* **Python** (Core logic)
* **Pandas/NumPy** (Data manipulation)
* **Scikit-Learn** (Machine Learning)
* **Streamlit** (Web Framework)
* **Pickle** (Model Deployment)
